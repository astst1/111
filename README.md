# Life Coach AI 应用

## 项目简介
这是一个基于DeepSeek R1 API开发的Life Coach AI应用。通过与AI进行对话，用户可以获得个人成长建议和指导。

## 技术架构
- 前端：HTML5 + CSS3，实现响应式设计的对话界面
- 后端：Node.js服务器，处理API请求和跨域问题
- API：DeepSeek R1 API，用于AI对话功能

## 功能特点
1. 实时对话：用户可以与AI进行实时对话交流
2. 流式响应：支持流式输出，提供更流畅的对话体验
3. 对话历史：保存并显示历史对话记录
4. 响应式设计：适配各种设备屏幕尺寸

## 项目结构
```
├── public/          # 静态资源目录
│   ├── css/         # 样式文件
│   └── js/          # JavaScript文件
├── server.js        # Node.js后端服务器
└── index.html       # 主页面
```

## 页面布局
主页面采用简洁的单页面设计：
- 顶部：标题栏
- 中部：对话历史显示区域
- 底部：用户输入框和发送按钮

## 开发规范
- 使用语义化HTML标签
- 采用Flexbox和Grid布局
- 添加详细的代码注释
- 确保代码符合W3C标准

## API配置
- 超时设置：60秒
- 温度参数：0.6
- 启用流式输出
